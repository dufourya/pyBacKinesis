# pyBacKinesis
Python code to run agent-based simulations of bacterial kinesis in dynamical environments.

Website: https://dufour.mmg.msu.edu

# Requirements
- python
- numpy
- scipy

Optional
- PyQt5
- PyQt5-sip
- pyqtgraph
- matplotlib

# Usage
python run_simulation.py

or

python run_animation.py
